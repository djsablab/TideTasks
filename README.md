
# ===== Tide Tasks =====

**Tide Tasks** is a powerful personal task scheduling application built with **React Native** with **Expo** environment. It's backend is heavily supported by **Firebase Database**, security of the authentications are backed up by **Firebase Authentication System**.

**TideTasks** offers a different approach for personal task scheduling with sleek and intuitive design while mattering the performance and stability.
